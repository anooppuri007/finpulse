# FinPulse

**Smart Financial Intelligence — India First**
by Anoop Puri | @theanooppuri

FinPulse is an AI-powered financial intelligence web app for Indian retail investors, traders, and students. It provides free, live, and comprehensive market data, acting as a powerful alternative to expensive platforms, built entirely using free tools and APIs.

## Features
- **Command Center:** Live updates for Nifty 50, Sensex, Bank Nifty, and global indices.
- **News + Sector Intelligence:** Filtered, curated news across 17 market sectors, with NLP sentiment scoring.
- **FII / DII Institutional Activity:** Accurate tracking of institutional flows from robust sources.
- **Economic Pulse & Market Events:** Live VIX, USD/INR, Yield curves, and official India Macro data.
- **PDF Reports:** Downloadable briefs of current market context.

## Deployment Details
- Built with **Streamlit**
- Designed for **Streamlit Cloud** deployment
- All times in **IST (UTC+5:30)**

## Setup
1. `pip install -r requirements.txt`
2. `streamlit run app.py`
