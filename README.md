## What FinPulse Does

### 📊 Core Intelligence (Original Features)
| Feature | Description |
|---|---|
| Live Market Snapshot | Global indices, commodities, crypto, currencies |
| Fear & Greed Index | Real-time market sentiment score (0–100) |
| AI-Filtered News | Top stories with 3-line summaries + source, click to expand |
| Reddit Pulse | Live sentiment from r/investing, r/wallstreetbets, r/stocks |
| Sector Watch | Hot and cold sectors connected to today's news |
| Contrarian Corner | The story nobody is talking about |
| Big Question of the Day | Analyst-level thinking prompt |
| AI Summary | Connects all dots in one paragraph |
| PDF Download | Full 7-page brief downloadable instantly |
| Auto-Refresh | Every 1 hour automatically |
| Manual Refresh | One-click fresh data anytime |

### ⚡ Aladdin Intelligence Suite (Advanced Features)
| Feature | What it does |
|---|---|
| **Technical Signal Scanner** | RSI, MACD, Bollinger Bands, MA Cross, Volume — Buy/Sell/Neutral signal for any global ticker |
| **Portfolio Intelligence Hub** | Track any global stocks — live P&L, Beta, diversification score, sector + regional allocation charts |
| **Economic Pulse** | VIX, Yield Curve (inverted = recession warning), Dollar Index, GDP growth (World Bank), Gold signal |
| **Earnings Radar** | Major earnings this week and next with live prices |
| **News → Market Impact** | Links each news story to the stocks it actually moves |
| **Asset Correlation Matrix** | Interactive heatmap — how 10 global assets move relative to each other |
| **Insider Activity** | SEC EDGAR Form 4 filings — who's buying and selling their own company's stock |

## ⚡ Run Locally
```bash
git clone https://github.com/theanooppuri/finpulse
cd finpulse
pip install -r requirements.txt
streamlit run app.py
```
